# Wizard — Persona

The section that shapes what the skill notices and prioritizes, built from lenses each grounded in a named authority or stated as an explicit directive.

## What a persona does

A persona shapes what the skill *notices* and *prioritizes* — implicit behavior specification through lenses, not explicit rules. "You are an expert in X" is an adjective, not a persona: adjectives don't change behavior, they just claim it.

---

## Lenses

Each lens is a distinct, non-overlapping analytical perspective. When grounded in a named authority or work, the format is:

```
**[Name/work]** gave you your [lens label]: [what it sees that others miss, and how it shapes behavior].
```

**Authorities** — a named real-world person whose body of work grounds a lens. Preferable to a generic descriptor ("an expert in X") because the model draws from that person's actual thinking, priorities, and blind spots. Load-bearing test: would swapping for another person in the same field change behavior? If no, too generic.

**Evaluating a lens** — ask: what does this perspective see that a generic model wouldn't? If you can't answer concretely, cut it. Coverage and distinctness matter; count alone is not a signal.

**Thin corpus** — if no well-known authority fits the domain, state lenses as explicit first-person directives instead. A longer, explicit persona beats a short one that relies on inference.

---

## Synthesis paragraph

The synthesis unifies the lenses into a single operating approach:

- Begins with "You work [approach]-first" or equivalent
- States the dominant operating approach and any hard constraints
- Names what this specialist never does

If it reads as a list of parallel behaviors rather than a coherent approach, the lenses may be too scattered. A slightly enumerative synthesis is acceptable if each lens is genuinely load-bearing.

---

## Axis relationship

Conversational skills need a richer Persona — the dialogue is where value is generated. Operational skills can carry a lighter one; the artifact spec does more work. Match persona depth to conversational weight.
